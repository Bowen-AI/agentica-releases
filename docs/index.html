Agentica

Private, open-weight agents on the computers you already own. Every Chat and Voice turn is agentic — tools, progress, and a final result.

Install macOS arm64 DMG Linux x64 AppImage

Latest builds publish to GitHub Releases. macOS builds are unsigned — see Gatekeeper notes below.

Install in one line

curl -fsSL https://agenticaai.vercel.app/install.sh | bash

Detects macOS or Linux, downloads from this repo’s latest release, and adds an agentica launcher. On Windows 11, use WSLg.

First run

  1. Install Ollama and pull a small model (ollama pull qwen3.5:4b-mlx).
  2. Chat — pick a workspace folder; ask the agent to create a file and verify it.
  3. Voice — allow the mic; first launch may download speech weights under ~/.local/share/agentica/voice.
  4. Jobs — draft a plan, submit to Local, watch status and logs.

macOS Gatekeeper

Builds are not notarized. If macOS blocks the app: right-click → Open, or System Settings → Privacy & Security → Open Anyway.

What’s included

  • Agentic Chat with independent model vs workspace machines (local / SSH).
  • Local voice: Silero VAD → Whisper → agent loop → Kokoro TTS.
  • Parallel tracked Jobs on local, SSH, or SLURM workers.
  • Deletable text History (no microphone audio stored).
