    Install

    One command downloads the latest build, clears Gatekeeper quarantine on macOS, starts Ollama, pulls qwen3.5:4b-mlx (~4 GB — keep ~5 GB free), and opens Agentica.

    curl -fsSL https://bowen-ai.github.io/agentica-releases/install.sh | bash

    Copied.

      What’s included

      Desktop app and bundled backend for Chat, Voice, Jobs, and History. After install, Chat works once the default model is on disk; voice weights download on first app start.

      • Default chat model: qwen3.5:4b-mlx (via install script; needs ~5 GB free)
      • Local voice: Whisper + Kokoro/Piper (auto on first app start, ~0.5 GB)
      • macOS builds are unsigned / not notarized — the one-liner clears quarantine
      • Linux x64: AppImage and tar.gz on the latest GitHub release
